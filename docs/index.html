FAQ

Short answers about Arcile.

Does Arcile collect any of my data?

No. Arcile does not request the android.permission.INTERNET permission, so it has no direct network access for telemetry, ads, or uploads.

Is Arcile free?

Yes. Arcile is free for personal use and source-available under the Tremors Source License (TSL). It does not include ads, trackers, or telemetry.

Can it work with SD cards and USB drives?

Yes. Arcile can browse internal storage, physical SD cards, and USB drives. Permanent removable volumes can also use dashboard, Recent Files, and Trash features.

Is Arcile on the Google Play Store?

Not currently. Arcile is distributed through GitHub Releases.

What Android versions are supported?

Arcile requires Android 11 or newer. It uses Android's all-files access permission for file management and notification permission for foreground operation updates.

How does the Trash system work?

On permanent volumes, Arcile can move deleted files into .arcile/.trash with metadata so they can be restored later.

Where should bugs and feature requests go?

Use the GitHub issue tracker. Include your device, Android version, and steps to reproduce when reporting bugs.
